# RedeemLoop EVM ERC-20 Voucher Example

This folder contains unaudited EVM example contracts for merchants that already use ERC-20 voucher assets.

These contracts are not a RedeemLoop asset issuance module and are not required by `packages/core`, `packages/sdk`, `services/api`, or the v0.2 PaymentIntent flow.

Included example contracts:

- `RedeemLoopERC20Voucher.sol`
- `MerchantVault.sol`
- `StoreTerminalRegistry.sol`
- `IRedeemLoopVoucher.sol`

Use these contracts only as local reference code for EVM projects. They are not audited and should not be deployed to mainnet without review.
