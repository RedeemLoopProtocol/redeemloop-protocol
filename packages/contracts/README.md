# RedeemLoop Contracts

This folder is a scaffold for the Solidity implementation. Phase 0 should implement:

- `RedeemLoopERC20Voucher.sol`
- `MerchantVault.sol`
- `StoreTerminalRegistry.sol`
- `IRedeemLoopVoucher.sol`

The first implementation should be ERC-20 based, with `decimals() = 0`, fixed max supply, and EIP-712 signature-based redemption.
